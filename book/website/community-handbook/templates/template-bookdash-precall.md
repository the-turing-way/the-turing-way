# HackMD template for pre-event call

*At least 1 week before the event, the core team organises the pre-event calls to share details about the program and clarify what other questions participants may have.
Notes from these calls can be collaboratively taken in HackMD using the template provided in this document.
The access permission can be set so that the notes can be read by everyone and edited by any signed-in users.**
